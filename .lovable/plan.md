

# خطة إعادة هيكلة وتحسين إعدادات العرض ومعاينة الفيديو

## المشاكل المكتشفة

بعد فحص `DisplaySettingsPanel.tsx` (795 سطر) و `VideoPreview.tsx` (2105 سطر):

1. **لوحة الإعدادات**: كل الخيارات في قائمة مسطحة طويلة بدون تجميع — صعبة التنقل
2. **أوضاع عرض الآيات** (`twoWords`, `threeTwo`, `wordByWord`): تستخدم `Date.now()` للتدوير مما يسبب عرض متقطع وغير متزامن مع الصوت — يجب أن تتبع `highlightedWordIndex`
3. **حجم الخط لأوضاع الكلمات**: نفس الحجم للكل — يجب أن تكون أكبر وأكثر سينمائية عند عرض كلمة أو كلمتين
4. **إعداد `textShadowStyle`**: موجود في الواجهة لكن **لا يُطبق أبداً** في Canvas — الظل مبرمج ثابت من `shadowIntensity`
5. **الجزيئات الذهبية**: الكود لا يزال موجوداً (سطر 1005-1088) رغم أن الإعداد `particleDensity` يكون `'off'` دائماً والواجهة لا تعرضه
6. **شارة رقم الآية**: موضعها ثابت `startY + totalHeight + 70*S` مما يسبب تداخل أو اختفاء في أوضاع الكلمات
7. **ترتيب الإعدادات**: غير منطقي — الأقسام المتعلقة ببعضها متفرقة

## خطة التنفيذ

### 1. إعادة هيكلة لوحة الإعدادات (`DisplaySettingsPanel.tsx`)

تنظيم الإعدادات في **5 أقسام قابلة للطي** (Accordion):

| القسم | المحتوى |
|-------|---------|
| **عناصر العرض** | أزرار إظهار/إخفاء (اسم السورة، القارئ، النص، الرقم) |
| **نمط النصوص** | شكل اسم السورة + موضعها، شكل اسم القارئ، ظل النص، تمييز الكلمات |
| **عرض الآيات** | طريقة عرض الآيات، شكل/لون رقم الآية، انتقال الآيات، الزخارف |
| **الإطار والخلفية** | إطار النص، انتقال الصور، وضع الأداء |
| **ابتهالات ومتقدم** | توهج/كلمات الابتهالات، علامة مائية، قوالب |

- إزالة `particleDensity` من الواجهة والـ interface نهائياً
- تبسيط وصف وضع الأداء

### 2. إصلاح عرض الآيات في الفيديو (`VideoPreview.tsx`)

**أ. حجم خط ديناميكي حسب وضع العرض:**
```
wordByWord → fontSize * 3.5 (كلمة واحدة كبيرة وسط الشاشة)
twoWords   → fontSize * 2.8
threeTwo   → fontSize * 2.4
full       → fontSize * 2.0 (الحالي)
```

**ب. إصلاح التزامن:**
- عند وجود `highlightedWordIndex` → متابعة دقيقة
- عند عدم وجوده → استخدام عداد ناعم بدل `Date.now()` مع fade بين الكلمات

**ج. تمركز عمودي محسّن:** الكلمات/المجموعات تظهر في وسط الشاشة تماماً مع هوامش مناسبة

### 3. تطبيق `textShadowStyle` فعلياً

قبل رسم نص الآية، ضبط ظل Canvas حسب الإعداد:
- `none`: `shadowBlur = 0`
- `soft`: `shadowBlur = 6*S`, alpha 0.4
- `strong`: `shadowBlur = 16*S`, alpha 0.8
- `glow`: `shadowBlur = 20*S`, لون ذهبي

### 4. تحسين شارة رقم الآية

- في وضع `full`: الموضع الحالي أسفل النص
- في أوضاع الكلمات: الشارة في منطقة ثابتة `canvas.height * 0.72` لمنع التداخل
- حجم الشارة يتناسب مع وضع العرض

### 5. تنظيف كود الجزيئات والنجوم

- إزالة كامل نظام الجزيئات (سطر 1005-1060) ونظام النجوم (1062-1088)
- إزالة `particlesRef` والمتغيرات المرتبطة
- هذا يقلل ~85 سطر ويحسن الأداء

### 6. تحسين الزخارف

- ضبط نقاط التحكم في الزخارف الجانبية لتتناسب مع `S`
- تحسين مواضع الفاصل العلوي
