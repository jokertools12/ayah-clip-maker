
# خطة إصلاح وتحسين منشئ فيديوهات قرآن ريلز

## ملخص المشاكل المطلوب حلها

1. **مشكلة حقول إدخال أرقام الآيات** - لا يمكن مسح أو تعديل الأرقام بشكل صحيح
2. **مشكلة أصوات القراء** - جميع القراء يشغلون نفس الصوت
3. **الفيديوهات المتحركة لا تظهر** - في المعاينة أو الفيديو المُنتج
4. **الخطوط كلها متشابهة** - لا يتم تطبيق الخط المختار
5. **التوهج الذهبي** - يجب أن يكون الافتراضي
6. **المكتبة لا تعمل** - التشغيل والتحميل لا يعملان
7. **مزيد من التحكمات** - أنماط عرض احترافية للآيات

---

## الحل التفصيلي

### 1. إصلاح حقول إدخال أرقام الآيات

**المشكلة:** الكود الحالي يمنع مسح القيمة لأنه يحولها فوراً إلى رقم

**الحل:**
- استخدام state منفصل للقيمة النصية (string) بدلاً من الرقمية
- السماح بحقل فارغ أثناء الكتابة
- تحويل القيمة إلى رقم فقط عند `onBlur` أو بعد تأخير

```text
┌─────────────────────────────────────────────────────┐
│  المنطق الحالي (خاطئ):                              │
│  onChange → parseInt فوراً → لا يسمح بالحذف        │
├─────────────────────────────────────────────────────┤
│  المنطق الجديد:                                     │
│  onChange → حفظ كـ string → onBlur → parseInt       │
└─────────────────────────────────────────────────────┘
```

**الملفات المتأثرة:** `src/pages/CreatePage.tsx`

---

### 2. إصلاح أصوات القراء المختلفين

**المشكلة:** Quran Foundation API قد يُرجع نفس recitationId لعدة قراء أو يفشل في التعرف عليهم

**الحل:**
- تحسين دالة `resolveRecitationIdByName` لتكون أكثر دقة
- إضافة mapping ثابت بين `reciter.id` المحلي و`recitationId` من Quran Foundation
- عند فشل Quran Foundation، استخدام mp3quran.net مباشرة مع server الخاص بكل قارئ

**الملفات المتأثرة:** 
- `src/lib/quranFoundationApi.ts` - إضافة mapping ثابت
- `src/data/reciters.ts` - إضافة `quranFoundationId` لكل قارئ
- `src/pages/PreviewPage.tsx` - تحسين منطق جلب الصوت

---

### 3. إصلاح الفيديوهات المتحركة (الخلفيات)

**المشكلة:** الفيديو لا يُرسم على Canvas لأن `videoReady` قد لا يتحقق أو CORS يمنع القراءة

**الحل:**
- استبدال روابط الفيديو الحالية بروابط موثوقة تدعم CORS
- التأكد من `crossOrigin="anonymous"` 
- إضافة تشخيص للفيديو (readyState)
- **خيار بديل:** استخدام صور متحركة GIF أو Lottie animations بدلاً من الفيديو

**الملفات المتأثرة:**
- `src/data/backgrounds.ts` - تحديث الروابط
- `src/components/VideoPreview.tsx` - تحسين التعامل مع الفيديو

---

### 4. إصلاح الخطوط العربية

**المشكلة:** Canvas يستخدم خط ثابت `"Noto Naskh Arabic"` بغض النظر عن الاختيار

**الحل:**
- تمرير `fontFamily` المختار من `textSettings` إلى Canvas
- التأكد من تحميل الخطوط من Google Fonts قبل الرسم
- استخدام `document.fonts.ready` للانتظار

**الملفات المتأثرة:**
- `src/components/VideoPreview.tsx` - استخدام `textSettings.fontFamily`
- `src/index.css` - التأكد من استيراد جميع الخطوط

---

### 5. جعل التوهج الذهبي هو الافتراضي

**التغيير:** تعديل القيمة الافتراضية لـ `highlightStyle` من `'solid'` إلى `'glow'`

**الملفات المتأثرة:**
- `src/pages/PreviewPage.tsx` - تغيير `DEFAULT_DISPLAY_SETTINGS`
- `src/components/VideoPreview.tsx` - تغيير `DEFAULT_DISPLAY_SETTINGS`

---

### 6. إصلاح صفحة المكتبة

**المشكلة الحالية:** الفيديوهات المحفوظة لا تحتوي على `video_url` حقيقي

**الحل:**
- عند الحفظ: تخزين معلومات الفيديو فقط (metadata)
- زر "تشغيل": يفتح صفحة المعاينة بنفس الإعدادات
- زر "تحميل": يظهر رسالة لإعادة إنشاء الفيديو

**الملفات المتأثرة:** `src/pages/LibraryPage.tsx`

---

### 7. إضافة أنماط عرض احترافية

**المميزات الجديدة:**
- أشكال زخرفية إسلامية حول النص
- إطارات متعددة للآيات (مستطيل، بيضاوي، زخرفي)
- أنماط ظهور متحركة (fade، slide، typewriter)

**الملفات المتأثرة:**
- `src/components/DisplaySettingsPanel.tsx` - إضافة خيارات جديدة
- `src/components/VideoPreview.tsx` - تنفيذ الأنماط الجديدة
