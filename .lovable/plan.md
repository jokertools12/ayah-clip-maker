

# 📖 موقع إنشاء مقاطع ريلز قرآنية

موقع احترافي عصري لإنشاء مقاطع فيديو قصيرة للقرآن الكريم مع تلاوات الشيوخ المشهورين وخلفيات طبيعية جميلة.

---

## 🎨 التصميم العام

- **تصميم عصري حديث** مع تدرجات لونية أنيقة (أزرق/أخضر/ذهبي)
- واجهة سهلة الاستخدام تدعم اللغة العربية بالكامل (RTL)
- تجاوب كامل مع جميع الأجهزة (موبايل، تابلت، ديسكتوب)
- أنيميشن ناعمة وانتقالات سلسة

---

## 📄 الصفحات الرئيسية

### 1. الصفحة الرئيسية
- عرض ترحيبي جذاب مع شرح الخدمة
- زر للبدء في إنشاء فيديو جديد
- عرض نماذج من الفيديوهات المنشأة

### 2. صفحة اختيار السورة
- قائمة بجميع سور القرآن الكريم (114 سورة)
- البحث السريع بالاسم أو الرقم
- تصنيف السور (مكية/مدنية)
- عرض عدد الآيات لكل سورة

### 3. صفحة إنشاء الفيديو (المحرر)
- **اختيار الشيخ/القارئ** من قائمة الشيوخ المشهورين
- **تحديد الآيات**: من آية X إلى آية Y
- **نوع الخلفية**: فيديوهات من Pexels أو صور مع تأثير Ken Burns
- **صيغة الفيديو**: عمودي (9:16) أو أفقي (16:9)
- **معاينة فورية** للآيات المختارة مع النص
- **تخصيص خط الآيات** (حجم ولون)

### 4. صفحة المعاينة والتصدير
- معاينة الفيديو قبل التصدير
- تحميل الفيديو بجودة عالية
- حفظ الفيديو في المكتبة الشخصية

### 5. صفحة المكتبة الشخصية (للمستخدمين المسجلين)
- عرض جميع الفيديوهات المحفوظة
- إعادة تحميل أو حذف الفيديوهات
- إحصائيات الاستخدام

### 6. صفحة تسجيل الدخول/التسجيل
- تسجيل دخول بالبريد الإلكتروني
- إمكانية الدخول كضيف (بدون حفظ)

---

## ⚙️ المميزات التقنية

### إنشاء الفيديو
- دمج الصوت مع الخلفية المرئية
- عرض نص الآية آية بآية متزامن مع الصوت
- نص عربي احترافي بخط واضح وجميل
- تأثيرات بصرية سلسة على النص

### مصادر البيانات (APIs مجانية)
- **القرآن الكريم**: AlQuran Cloud API
- **الصوتيات**: MP3Quran API للشيوخ المشهورين
- **الخلفيات**: Pexels API للفيديوهات والصور الطبيعية

### القراء المتاحون
- عبد الباسط عبد الصمد
- مشاري العفاسي
- ماهر المعيقلي
- سعود الشريم
- عبد الرحمن السديس
- وغيرهم من القراء المشهورين

---

## 🗄️ نظام المستخدمين (Backend)

- **نظام المصادقة**: تسجيل دخول وإنشاء حساب
- **حفظ الفيديوهات**: تخزين الفيديوهات المنشأة
- **سجل الإنشاء**: تاريخ الفيديوهات التي أنشأها المستخدم

---

## 🚀 مميزات إضافية

- **مشاركة سريعة**: زر مشاركة مباشر لوسائل التواصل
- **المفضلة**: حفظ السور المفضلة للوصول السريع
- **الوضع المظلم/الفاتح**: تبديل بين المظهرين
- **إحصائيات**: عدد الفيديوهات المنشأة والتحميلات
